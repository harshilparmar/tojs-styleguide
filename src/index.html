Brand Manual & Style Guide

Version 2.0

The Toronto JS Brand Manual & Style Guide is a living document that describes the style we use at Toronto JS. It is intended to be a reference for current and future contributors to the Toronto JS community who wish to develop or create anything using Toronto JS branding.

Using the fonts ðŸ”—

Montserrat - Medium/extra bold

Montserrat is the main font. Itâ€™s intended use is for body text and headers.

When paired with other bold text like the "JS" from Toronto JS' logo it should be used in medium weight to create visual contrast with the pairing text. The medium weight is also to be used for "main headings", where sufficient size difference exists between the body text and header.

For headers that are too close to text and the size difference is not enough to represent a "break" between text and header, the extra bold weight should be used.

On body text italic and bold text can be used to create emphasis when needed.

Source Sans Pro - Black

Used only for small details, like the "JS" in the Toronto JS logo or places where it will be the only piece of text present. This font intent is to draw attention to the text.

Source Code Pro - Regular
Used for code blocks and subtle emphasis on specific words and phrases. Like the "online" or "in person" mark on an event art.

Colour usage ðŸ”—

he main colour is "Toronto JS Red", it has two accent shades, one darker ("Breaking news") and one lighter ("CN Tower lights"). Those are the colours to be used in most contexts to highlight some details or make things "pop".

When using against a dark background the preferred pairing is the main colour follower by the lighter accent. Whereas the pairing for a light background should be the opposite, main colour followed by the darker accent.

There are also four neutral shades to be used, a black one ("Racoon stripes"), a white one ("Winter sky") and two greys ("Downtown street" and "Slushy snow").The black and white shades are to be used mainly for backgrounds and the basis of light and dark themes.The shades of grey serve to add highlights, depth or details to the other elements they are paired with.

Logo Anatomy ðŸ”—

CN Tower

The CN Tower is one of the iconic landmarks of Toronto, for this reason it was chosen to be the symbol to represent the Toronto JS community.

JS

Easely recognized by developers, the yellow square with the letters "JS" are known as the "JS Logo".

Even it not being official, this easy recognision serves as inspiration for a community based around Javascript developers.

Logo Variants & Uses ðŸ”—

Regular Logo

This is the main version to be used for most porpouses, as most of our uses are digital it is preffered to draw attention to the logo and related content.

Monochrome Logo

When the regular logo is not suitable, the monochrome version can be used instead.

It is preffered to use the monochrome version for watermarks or other places where color is a constraint.

Dark Contrast Logo

It is preffered to use the dark contrast version for dark backgrounds where the regular logo would be unsuitable or draw too much attention to the logo instead of the content.

Light Contrast Logo

It is preffered to use the light contrast version for light backgrounds where the regular logo would be unsuitable or draw too much attention to the logo instead of the content.

Icon

A simplified version of the logo, it is the preffered version for small spaces where the regular logo would make the details be lost or "JS" be illegible.

Pictures & Art ðŸ”—

Pictures ðŸ”—

Photos and other pictures used for events and promotion should always be well lit, display action and try to portrait people in a diverse and inclusive way. Display of brands should be avoided.

Art ðŸ”—

Drawings and other art used is advised to be simple in style so it can be more easely recognized at different sizes and have higher fidelity if printed. It is also suggested to keep to warm tones preferably red and the brand colors.

AI generated art is welcome.

Graphic Material ðŸ”—

There are two main styles to be used: one has the CN Tower positioned to one of the sized and the other has only a part of the CN Tower logo.

The second style is for composing multiple things together, like a group of events. So when we join all items it comes together to form the CN Tower image.

Aside from those two styles, there are other two extras for printed material, they are best suited for application as watermarks or situations where colour use is a restriction.
